# CRUD de Alunos - Projeto Python

Estrutura de projeto:
- src/main.py
- src/services.py


Funcionalidades:
- Cadastrar aluno (Create)
- Listar alunos (Read)
- Atualizar aluno (Update)
- Excluir aluno (Delete)
- Matrícula gerada automaticamente: CURSO + número sequencial (ex: GES1, GES2)
